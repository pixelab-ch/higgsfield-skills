## Master Template

Use this structure as the foundation for all product-360 videos:

```
[OPENING HOOK - 2 seconds]
Product Type: [category]
Hero Angle: [specific angle and lighting setup]
Hook Style: [particle / spotlight / crash / unwrap / macro-to-wide / assembly /
             exploded / liquid / lift / spin / zoom / halo]
Mood: [dramatic / elegant / energetic / serene / playful]

[HERO SHOWCASE - 3-5 seconds]
Rotation Speed: [slow / medium / fast]
Primary Rotation: [full 360 / half orbit / floating / hero lock]
Lighting: [key light angle / fill intensity / rim light presence]
Camera: [static / subtle orbit / push-in / pull-back / tilt]
Focus on: [main feature / material / proportion / detail]

[DETAIL PROGRESSION - 3-5 seconds]
Macro Shots: [specific textures / stitching / engravings / material close-ups]
Transition: [push-in / dissolve / spin-cut / morph]
Reveal: [hidden detail / underside / interior / feature callout]

[LIFESTYLE / CONTEXT - 2-3 seconds] (optional for product page videos)
Environment: [studio / lifestyle / lifestyle with hands / packaged]
Lighting Shift: [to warm / to natural / to product-focused]
Narrative: [in-use / luxury presentation / gifting moment]

[FINALE - 2 seconds]
Return to: [hero angle / product in light / brand moment]
Final Effect: [light flare / slow fade / gentle spin / freeze]
Text Overlay: [product name / key spec / price / CTA]
```

## Example 1 — Luxury Watch Turntable (22 lines)

```
Product Category: Luxury Wristwatch (Swiss-Made, Mechanical Movement)
Hook Style: Dramatic Spotlight Snap-On with Halo Light Flare
Duration: 25 seconds | Aspect Ratio: 16:9

Opening (0–2 s):
Darkness. Single harsh spotlight snaps on, illuminating a luxury Swiss watch in dramatic
isolation. Watch gleams in hard light, reflection catches on polished steel case. Brief pause
to emphasize craftsmanship. Halo light flare cascades across sapphire crystal.

Hero Showcase (2–8 s):
Watch rotates slowly 360° at 35° angle, emphasizing front dial face first. Dial shows
perfect legibility: Roman numerals, hand details, date window. Cool lighting (5400K)
emphasizes steel precision. Specular highlights dance across polished case. Subtle fill
maintains shadow detail. Rotation reveals right profile showing case thickness, lugs, crown.

Rotation: 12 s per full 360° (slow). Lighting: single hard key 45° above-left, subtle fill,
strong rim.

Detail Progression (8–16 s):
Pull-in macro to reveal dial texture — concentric circles, printing perfection, hand finishing.
Push-out to caseback: exhibition showing mechanical movement, engravings, serial number.
Macro on case side profile emphasizing brushed finish grain, lugs, spring bar detail.

Finale (16–25 s):
Return to hero angle. Lighting shifts warmer (3500K) to emphasize gold accents on dial
markers. Final gentle rotation (3 s, slow). Text overlay: "Premium Automatic Movement" +
"Swiss-Made" + "From $8,500". Final light flare on crystal. Fade to black.
```

## Example 2 — Performance Sneaker Reveal (24 lines)

```
Product Category: High-Performance Athletic Sneaker
Hook Style: Unwrap Peel Reveal + Crash & Shockwave
Duration: 18 seconds | Aspect Ratio: 1:1

Opening (0–2 s):
Translucent protective plastic slowly peels back from pristine white sneaker. As plastic
retracts, shoe emerges — pure, unboxed, fresh. Brief freeze-frame at pinnacle of reveal.
Shoe drops onto white surface from above, subtle shockwave ripples outward underneath.

Hero Showcase (2–8 s):
Sneaker rotates on white platform, starting at 45° emphasizing sleek profile. Camera orbits
subtly (parallax effect) while shoe rotates, creating 3D depth. Warm lighting (3500K) from
above emphasizes upper mesh texture. Rotation reveals all sides: left profile, back (heel
and branding), right profile, front (toe box taper). Medium rotation (6 s per 360°).
Fill light from underneath catches sole tread — sharp, defined, performance-oriented.

Detail Progression (8–14 s):
Macro push-in on heel counter showing structural detail and cushioning callout. Quick
transition: pull-back to full shoe, then push-in on toe box (reinforcement and mesh pattern).
Close-up on sole: tread pattern clarity, material texture. Overhead: arch support and branding.

Finale (14–18 s):
Return to three-quarter angle. Lighting shift emphasizes color gradients. Final slow
rotation (2 s) with gentle upward camera tilt, elevating the shoe. Text overlay:
"Next-Gen Performance" + "Carbon Plate Technology" + "Available Now". Fade to brand logo.
```

## Example 3 — Premium Wireless Earbuds (25 lines)

```
Product Category: Premium Wireless Audio Earbuds + Charging Case
Hook Style: Particle Materialization + Self-Assembly
Duration: 20 seconds | Aspect Ratio: 16:9

Opening (0–2 s):
Darkness. Golden particles float and swirl, gradually coalescing. Particles converge and
solidify, forming the sleek charging case first. Case settles. Particles continue to assemble
into individual earbuds, floating above case. Earbuds snap into place. Satisfying click.

Hero Showcase (2–8 s):
Case rotates slowly 360° on white surface, emphasizing industrial design. Cool, precise
lighting (5600K) with sharp specular highlights on glossy finish. Rotation reveals all
faces: top (LED indicator), sides, back (charging port). Camera angle: 25° overhead to
emphasize geometry and premium finish. Medium rotation (7 s per 360°). Subtle fill maintains
detail in recessed areas.

Detail Progression (8–15 s):
Push-in macro to earbud fit — show in-ear design, earpieces, material (silicone or titanium).
Reveal case lid opening — earbud placement inside, magnetic positioning, clean interior.
Close-up on charging port: USB-C detail. Macro on touch-sensitive surface. Overhead footprint.

Interactive Showcase (15–18 s):
Hand placing earbud in ear, showing scale and fit. Case nestled in palm, emphasizing
portability. Return to case closed, resting on neutral surface.

Finale (18–20 s):
Case rotates final time, slower pace. Text overlays: "40-Hour Battery" + "Premium Noise
Cancellation" + "Starting at $199". Final light catch on glossy finish. Fade to clean white.
```

## Example 4 — Luxury Fragrance Beauty Shot (23 lines)

```
Product Category: Premium Fragrance Bottle + Luxury Cosmetic Compact
Hook Style: Halo Light Flare + Liquid Pour Transparency
Duration: 22 seconds | Aspect Ratio: 16:9

Opening (0–2 s):
Soft, warm light reveals a crystal perfume bottle in minimalist setup. Brilliant light flare
cascades across glass surface, emphasizing clarity and luxury. Brief lingering moment on
radiant light reflection. Bottle slowly tilts, golden liquid shifts inside, catching light.

Hero Showcase (2–9 s):
Bottle rotates slowly on white surface, front-facing initially. Warm, diffuse lighting
(3200K) creates soft aspirational glow. No harsh shadows. Rotation reveals all angles: front
label detail, side (liquid color), back (branding and engraving). Emphasize: glass clarity,
golden liquid saturation, stopper detail. Slow rotation (10 s per 360°). Subtle back
lighting separates bottle without harsh rim.

Detail Progression (9–16 s):
Macro zoom into stopper — crystal faceting or carved detail. Pull-back to full bottle, then
push-in on label: typography, branding, engraving detail. Close-up on liquid inside: color
gradient and transparency. Overhead: stopper geometric precision, luxury craft. Macro on cap.

Lifestyle Context (16–19 s):
Bottle positioned on marble with complementary items (delicate flowers, silk scarf). Hand
gently touches bottle to show scale and invite intimate interaction. Soft-focus editorial feel.

Finale (19–22 s):
Return to hero angle — bottle isolated on white, pure and elegant. Lighting shifts warmer and
softer. Text overlay: product name + "Eau de Parfum" + fragrance notes + "From $120". Final
light flare on glass. Gentle fade to soft white.
```

## Example 5 — Premium Leather Car Seat Covers (26 lines)

```
Product Category: Luxury Leather Car Seat Covers (Full Set: Driver + Passenger)
Hook Style: Dramatic Spotlight Reveal + Multi-Speed Rotation
Duration: 28 seconds | Aspect Ratio: 16:9

Opening (0–2.5 s):
Darkness. Single dramatic spotlight slowly illuminates pristine leather seat cover draped
over a form. Light catches stitching detail and grain as spotlight intensity increases.
Leather gleams with subtle warmth. Brief pause emphasizing premium craftsmanship. Rotation
begins, slowly accelerating.

Hero Showcase (2.5–12 s):
Complete set displayed on black form (simulating car interior). Rotation starts slow (12 s
per 360°), emphasizing every stitch and material detail. Warm, dramatic lighting (3200K)
from 45° above. Hard rim light separates from background. Rotation reveals: front cover,
side profile (thickness and drape), stitching patterns, seams, embossing. Slight overhead
to show form-fitting detail and quilted pattern. Mid-rotation (10 s): speed accelerates
slightly (7 s remaining 180°) to add energy.

Material Deep-Dive (12–18 s):
Rotation pauses on hero angle (driver's side, 3/4 view). Push-in macro to stitching: show
precision, color coordination, professional finish. Pull-back, then zoom to leather grain:
texture, color depth, premium quality. Close-up on embossing or quilting: geometric
precision and luxury design. Macro on edge stitching: secure, elegant finishing.

Context and Functionality (18–24 s):
Seat covers repositioned on actual car seat (or realistic mock-up). Show fit, contour, and
integration with car interior. Multiple angles: driver's comfort position, installation hints.
Lifestyle: driver hand on steering wheel with covered seat visible.

Finale (24–28 s):
Full set shot: both driver and passenger covers, symmetrical composition. Return to
soft-dramatic lighting. Slow final rotation emphasizing leather perfection. Text overlay:
"Premium Napa Leather" + "Custom Quilted Design" + "Installation Included" + "From $599".
Final light catch on leather surface. Fade through warm tone.
```

## Common Mistakes and Fixes

| Mistake | Fix |
|---------|-----|
| Slow rotation with no detail engagement | Vary rotation speed; add macro detail shots every 5–7 s |
| Background busier than product | Use neutral gradient; ensure contrast ratio between product and background |
| Inconsistent lighting across multi-angle images | Normalize lighting; use dissolves to hide shifts |
| Text overlay too small or low-contrast | Minimum 24 px body text; 2 s minimum display time |
| Rotation jitter or uneven motion | Ensure smooth frame interpolation; consistent frame rate |
| Hook mismatched to product category | Luxury = elegant hooks; tech = energetic; fashion = lifestyle |
| No macro detail shots | Always include 2–5 s of macro (stitching, grain, engraving) |
| Lifestyle context forced or irrelevant | Match lifestyle to "how do I use this?" or "where does this belong?" |
| No call-to-action or next step | Final text overlay: "Shop Now," "Learn More," "Link in Bio," price |
| Product too small in frame | Product should occupy 50–80% of frame throughout |
