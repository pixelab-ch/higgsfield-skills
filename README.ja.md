# これは何ですか？ What Is This?

Seedance 2.0（Higgsfield）専用の15個の専門的なClaudeスキル。各スキルは、Claudeを特定のビデオスタイルのプロンプトエンジニアに変え、大規模で詳細で直接使用可能なプロンプトを生成します。強力な**2秒フック**を含み、視聴者のスクロールを止めます。

15 specialized Claude skills built for Seedance 2.0 on Higgsfield. Each skill turns Claude into a prompt engineer for a specific video style — generating large, detailed, paste-ready prompts with powerful **2-second hooks** that stop the scroll.

👉 **[Higgsfield でビデオを作成を開始する Start creating on Higgsfield](https://higgsfield.ai/create/video?model=seedance_2_0)**

---

## 各スキルに含まれるもの What Each Skill Contains

- **2秒フックフレームワーク / 2-Second Hook Framework** — 10-12種類の注目を集めるオープニングパターン | 10-12 attention-grabbing opener patterns
- **タイムラインセグメンテーション / Timeline Segmentation** — ビート単位の分解、最大15秒 | Beat-by-beat breakdown up to 15s
- **カメラムーブメント百科事典 / Camera Movement Encyclopedia** — 15-20以上のテクニックと正確なプロンプト表現 | 15-20+ techniques with exact phrasing
- **ライティングと雰囲気 / Lighting & Atmosphere** — ムードと品質を表現するライティング設定 | Setups that communicate mood and quality
- **サウンドデザイン / Sound Design** — アンビエント、フォーレイ、ミュージック、サイレンス | Ambient, foley, music, silence
- **マテリアル参照戦略 / Material Reference Strategy** — `@image1` `@video1` `@audio1` のベストプラクティス | Best practices for references
- **プラットフォーム最適化 / Platform Optimization** — Douyin、TikTok、Instagram、YouTube など | Cross-platform adjustments
- **5以上の大規模プロンプト例 / 5+ Large Example Prompts** — 各15-25行、プロダクションレベルの品質 | 15-25 lines each, production-quality

## 🚀 クイックスタート Quick Start

### スキルをインストール Install a Skill

1. `SKILL.md` をClaudeスキルディレクトリにコピーします | Copy `SKILL.md` to your Claude skills directory
2. 作成したいビデオについて説明します | Describe the video you want to create
3. Claudeがプロダクションレベルのプロンプトを生成します | Claude generates a production-ready prompt
4. [Seedance 2.0（Higgsfield）](https://higgsfield.ai/create/video?model=seedance_2_0)に貼り付けます | Paste into Seedance 2.0 on Higgsfield with your materials

### 例 Example

```
あなた: 15秒のシネマティックビデオが必要です。夜明けの霧がかかった竹林を歩く孤独な侍
You: I need a 15s cinematic video of a lone samurai walking through foggy bamboo forest at dawn

Claude（シネマティックスキルを使用 with cinematic skill）:
→ 25行以上の詳細なプロンプトを生成します Generates 25+ line detailed prompt
→ タイムライン、カメラ、ライティング、サウンド、2秒フックを含みます With timeline, camera, lighting, sound, 2-second hook
```

## 📋 Seedance 2.0（Higgsfield）仕様 Specs

| 入力 Input | フォーマット Format | 制限 Limit |
|---|---|---|
| 画像 Image | jpeg, png, webp, bmp, tiff, gif | ≤ 9個、各 < 30MB |
| ビデオ Video | mp4, mov | ≤ 3個、各 < 50MB、総時間 2–15s |
| オーディオ Audio | mp3, wav | ≤ 3個、各 < 15MB、総時間 ≤ 15s |
| テキスト Text | 自然言語 Natural language | — |
| **合計 Combined** | — | **≤ 12個のファイル files** |
| **出力 Output** | ビデオ Video | **4–15s, 720p、サウンド含む with sound** |
