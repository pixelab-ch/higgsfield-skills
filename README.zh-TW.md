# 這是什麼？ What Is This?

15 個專業 Claude 技能，專為 Seedance 2.0（Higgsfield）打造。每個技能將 Claude 變成特定影片風格的提示詞工程師——生成大型、詳細、可直接使用的提示詞，包含強力 **2 秒鉤子**，讓觀眾停止滑動。

15 specialized Claude skills built for Seedance 2.0 on Higgsfield. Each skill turns Claude into a prompt engineer for a specific video style — generating large, detailed, paste-ready prompts with powerful **2-second hooks** that stop the scroll.

👉 **[立即在 Higgsfield 上建立影片 Start creating on Higgsfield](https://higgsfield.ai/create/video?model=seedance_2_0)**

## 每個技能包含 What Each Skill Contains

- **2 秒鉤子框架 / 2-Second Hook Framework** — 10-12 種抓取注意力的開場模式 | 10-12 attention-grabbing opener patterns
- **時間線分段 / Timeline Segmentation** — 逐拍分解，最長 15 秒 | Beat-by-beat breakdown up to 15s
- **攝影機運動百科 / Camera Movement Encyclopedia** — 15-20+ 技術及精確提示詞 | 15-20+ techniques with exact phrasing
- **燈光與氛圍 / Lighting & Atmosphere** — 傳達情緒和品質的布光方案 | Setups that communicate mood and quality
- **聲音設計 / Sound Design** — 環境音、擬音、音樂、靜默 | Ambient, foley, music, silence
- **素材引用策略 / Material Reference Strategy** — `@image1` `@video1` `@audio1` 的最佳用法 | Best practices for references
- **平台最佳化 / Platform Optimization** — 抖音、TikTok、Instagram、YouTube 等 | Cross-platform adjustments
- **5+ 大型示例提示詞 / 5+ Large Example Prompts** — 每個 15-25 行，製作級品質 | 15-25 lines each, production-quality

## 🚀 快速開始 Quick Start

### 安裝技能 Install a Skill

1. 將 `SKILL.md` 複製到 Claude 技能目錄 | Copy `SKILL.md` to your Claude skills directory
2. 描述你想建立的影片 | Describe the video you want to create
3. Claude 生成製作級提示詞 | Claude generates a production-ready prompt
4. 貼到 [Seedance 2.0（Higgsfield）](https://higgsfield.ai/create/video?model=seedance_2_0) | Paste into Seedance 2.0 on Higgsfield with your materials

### 示例 Example

```
你: 我需要一個 15 秒電影風格影片，孤獨武士在黎明霧氣竹林中行走
You: I need a 15s cinematic video of a lone samurai walking through foggy bamboo forest at dawn

Claude (使用電影技能 with cinematic skill):
→ 生成 25+ 行詳細提示詞 Generates 25+ line detailed prompt
→ 包含時間線、鏡頭、燈光、聲音、2 秒鉤子 With timeline, camera, lighting, sound, 2-second hook
```

| **輸出 Output** | 影片 Video | **4–15s, 720p, 含聲音 with sound** |
