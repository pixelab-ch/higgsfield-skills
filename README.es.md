# ¿Qué es esto? What Is This?

15 habilidades especializadas de Claude creadas específicamente para Seedance 2.0 (Higgsfield). Cada habilidad convierte a Claude en un ingeniero de prompts para un estilo de video específico, generando prompts grandes, detallados y listos para usar con poderosos **ganchos de 2 segundos** que detienen el desplazamiento.

15 specialized Claude skills built for Seedance 2.0 on Higgsfield. Each skill turns Claude into a prompt engineer for a specific video style — generating large, detailed, paste-ready prompts with powerful **2-second hooks** that stop the scroll.

## Lo que contiene cada habilidad What Each Skill Contains

- **Marco de gancho de 2 segundos / 2-Second Hook Framework** — 10-12 patrones de apertura que atraen la atención | 10-12 attention-grabbing opener patterns
- **Segmentación de línea de tiempo / Timeline Segmentation** — Desglose beat por beat, hasta 15s | Beat-by-beat breakdown up to 15s
- **Enciclopedia de movimiento de cámara / Camera Movement Encyclopedia** — 15-20+ técnicas con frases exactas de prompts | 15-20+ techniques with exact phrasing
- **Iluminación y atmósfera / Lighting & Atmosphere** — Configuraciones de iluminación que comunican estado de ánimo y calidad | Setups that communicate mood and quality
- **Diseño de sonido / Sound Design** — Ambiente, foley, música, silencio | Ambient, foley, music, silence
- **Estrategia de referencia de material / Material Reference Strategy** — Mejores prácticas para `@image1` `@video1` `@audio1` | Best practices for references
- **Optimización de plataforma / Platform Optimization** — Douyin, TikTok, Instagram, YouTube, etc. | Cross-platform adjustments
- **5+ prompts de ejemplo grandes / 5+ Large Example Prompts** — 15-25 líneas cada uno, calidad de producción | 15-25 lines each, production-quality

→ Con línea de tiempo, cámara, iluminación, sonido, gancho de 2 segundos With timeline, camera, lighting, sound, 2-second hook

## 📋 Especificaciones de Seedance 2.0 (Higgsfield) Specs

| Entrada Input | Formato Format | Límite Limit |
|---|---|---|
| Imagen Image | jpeg, png, webp, bmp, tiff, gif | ≤ 9, < 30MB cada una |
| Video | mp4, mov | ≤ 3, < 50MB cada una, duración total 2–15s |
| Audio | mp3, wav | ≤ 3, < 15MB cada una, duración total ≤ 15s |
| Texto Text | Lenguaje natural Natural language | — |
| **Combinado Combined** | — | **≤ 12 archivos files** |
| **Salida Output** | Video | **4–15s, 720p, con sonido with sound** |
