# C'est quoi ? What Is This?

15 compétences Claude spécialisées conçues pour Seedance 2.0 (Higgsfield). Chaque compétence transforme Claude en ingénieur de prompts pour un style vidéo spécifique — générant des prompts volumineux, détaillés et prêts à coller avec un puissant **crochet de 2 secondes** qui arrête le défilement.

15 specialized Claude skills built for Seedance 2.0 on Higgsfield. Each skill turns Claude into a prompt engineer for a specific video style — generating large, detailed, paste-ready prompts with powerful **2-second hooks** that stop the scroll.

👉 **[Commencez à créer sur Higgsfield Start creating on Higgsfield](https://higgsfield.ai/create/video?model=seedance_2_0)**

---

## Ce que contient chaque compétence What Each Skill Contains

- **Cadre de Crochet de 2 Secondes / 2-Second Hook Framework** — 10-12 modèles d'ouverture captivants | 10-12 attention-grabbing opener patterns
- **Segmentation de la Chronologie / Timeline Segmentation** — Décomposition battement par battement, jusqu'à 15s | Beat-by-beat breakdown up to 15s
- **Encyclopédie des Mouvements de Caméra / Camera Movement Encyclopedia** — 15-20+ techniques avec formulation précise | 15-20+ techniques with exact phrasing
- **Éclairage et Atmosphère / Lighting & Atmosphere** — Configurations d'éclairage qui transmettent l'humeur et la qualité | Setups that communicate mood and quality
- **Conception Sonore / Sound Design** — Ambiance, bruitages, musique, silence | Ambient, foley, music, silence
- **Stratégie de Référence Matérielle / Material Reference Strategy** — Meilleures pratiques pour `@image1` `@video1` `@audio1` | Best practices for references
- **Optimisation Multiplateforme / Platform Optimization** — Ajustements pour Douyin, TikTok, Instagram, YouTube, etc. | Cross-platform adjustments
- **5+ Grands Exemples de Prompts / 5+ Large Example Prompts** — 15-25 lignes chacun, qualité de production | 15-25 lines each, production-quality

## 🚀 Démarrage Rapide Quick Start

### Installer une Compétence Install a Skill

1. Copiez `SKILL.md` dans votre répertoire de compétences Claude | Copy `SKILL.md` to your Claude skills directory
2. Décrivez la vidéo que vous souhaitez créer | Describe the video you want to create
3. Claude génère un prompt prêt pour la production | Claude generates a production-ready prompt
4. Collez dans [Seedance 2.0 (Higgsfield)](https://higgsfield.ai/create/video?model=seedance_2_0) | Paste into Seedance 2.0 on Higgsfield with your materials

### Exemple Example

```
Vous: J'ai besoin d'une vidéo cinématique de 15 secondes d'un samuraï solitaire marchant dans une forêt de bambous brumeuse à l'aube
You: I need a 15s cinematic video of a lone samurai walking through foggy bamboo forest at dawn

Claude (utilisant la compétence cinématique with cinematic skill):
→ Génère un prompt détaillé de 25+ lignes Generates 25+ line detailed prompt
→ Avec chronologie, caméra, éclairage, son, crochet de 2 secondes With timeline, camera, lighting, sound, 2-second hook
```

## 📋 Specs Seedance 2.0 (Higgsfield) Specs

| Entrée Input | Format Format | Limite Limit |
|---|---|---|
| Image Image | jpeg, png, webp, bmp, tiff, gif | ≤ 9, chacun < 30MB |
| Vidéo Video | mp4, mov | ≤ 3, chacun < 50MB, durée totale 2–15s |
| Audio Audio | mp3, wav | ≤ 3, chacun < 15MB, durée totale ≤ 15s |
| Texte Text | Langage naturel Natural language | — |
| **Combiné Combined** | — | **≤ 12 fichiers files** |
| **Sortie Output** | Vidéo Video | **4–15s, 720p, avec son with sound** |
