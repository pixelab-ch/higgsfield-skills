# Was ist das? What Is This?

15 spezialisierte Claude-Fähigkeiten, speziell für Seedance 2.0 (Higgsfield) entwickelt. Jede Fähigkeit verwandelt Claude in einen Prompt-Engineer für einen bestimmten Videostil und generiert große, detaillierte, sofort einsatzbereit Prompts mit starken **2-Sekunden-Hooks**, die zum Scrollen verleiten.

15 specialized Claude skills built for Seedance 2.0 on Higgsfield. Each skill turns Claude into a prompt engineer for a specific video style — generating large, detailed, paste-ready prompts with powerful **2-second hooks** that stop the scroll.

## Was jede Fähigkeit beinhaltet What Each Skill Contains

- **2-Sekunden-Hook-Framework / 2-Second Hook Framework** — 10-12 aufmerksamkeitserregende Öffnungsmuster | 10-12 attention-grabbing opener patterns
- **Zeitleisten-Segmentierung / Timeline Segmentation** — Beat-für-Beat-Aufschlüsselung, bis zu 15s | Beat-by-beat breakdown up to 15s
- **Kamerabewegungs-Enzyklopädie / Camera Movement Encyclopedia** — 15-20+ Techniken mit genauen Prompt-Formulierungen | 15-20+ techniques with exact phrasing
- **Beleuchtung und Atmosphäre / Lighting & Atmosphere** — Beleuchtungssetups, die Stimmung und Qualität vermitteln | Setups that communicate mood and quality
- **Sounddesign / Sound Design** — Ambient, Foley, Musik, Stille | Ambient, foley, music, silence
- **Material-Referenzstrategie / Material Reference Strategy** — Best Practices für `@image1` `@video1` `@audio1` | Best practices for references
- **Plattformoptimierung / Platform Optimization** — Douyin, TikTok, Instagram, YouTube usw. | Cross-platform adjustments
- **5+ große Beispiel-Prompts / 5+ Large Example Prompts** — je 15-25 Zeilen, Produktionsqualität | 15-25 lines each, production-quality

1. Kopieren Sie `SKILL.md` in Ihr Claude-Fähigkeitsverzeichnis | Copy `SKILL.md` to your Claude skills directory
2. Beschreiben Sie das Video, das Sie erstellen möchten | Describe the video you want to create
3. Claude generiert einen produktionsreifen Prompt | Claude generates a production-ready prompt
4. Fügen Sie in [Seedance 2.0 (Higgsfield)](https://higgsfield.ai/create/video?model=seedance_2_0) ein | Paste into Seedance 2.0 on Higgsfield with your materials

```
→ Generiert 25+ Zeilen detaillierter Prompts Generates 25+ line detailed prompt
→ Mit Zeitleiste, Kamera, Beleuchtung, Sound, 2-Sekunden-Hook With timeline, camera, lighting, sound, 2-second hook
```
