# O que é isso? What Is This?

15 habilidades Claude especializadas projetadas para Seedance 2.0 (Higgsfield). Cada habilidade transforma Claude em um engenheiro de prompts para um estilo de vídeo específico — gerando prompts grandes, detalhados e prontos para colar com um poderoso **gancho de 2 segundos** que para a rolagem.

15 specialized Claude skills built for Seedance 2.0 on Higgsfield. Each skill turns Claude into a prompt engineer for a specific video style — generating large, detailed, paste-ready prompts with powerful **2-second hooks** that stop the scroll.

👉 **[Comece a criar no Higgsfield Start creating on Higgsfield](https://higgsfield.ai/create/video?model=seedance_2_0)**

---

## O que cada habilidade contém What Each Skill Contains

- **Estrutura de Gancho de 2 Segundos / 2-Second Hook Framework** — 10-12 padrões de abertura que capturam atenção | 10-12 attention-grabbing opener patterns
- **Segmentação da Linha do Tempo / Timeline Segmentation** — Decomposição batida por batida, até 15s | Beat-by-beat breakdown up to 15s
- **Enciclopédia de Movimento de Câmera / Camera Movement Encyclopedia** — 15-20+ técnicas com fraseado exato | 15-20+ techniques with exact phrasing
- **Iluminação e Atmosfera / Lighting & Atmosphere** — Configurações que comunicam humor e qualidade | Setups that communicate mood and quality
- **Design de Som / Sound Design** — Ambiente, efeitos sonoros, música, silêncio | Ambient, foley, music, silence
- **Estratégia de Referência de Materiais / Material Reference Strategy** — Melhores práticas para `@image1` `@video1` `@audio1` | Best practices for references
- **Otimização de Plataforma / Platform Optimization** — Ajustes para Douyin, TikTok, Instagram, YouTube, etc. | Cross-platform adjustments
- **5+ Exemplos de Prompts Grandes / 5+ Large Example Prompts** — 15-25 linhas cada um, qualidade de produção | 15-25 lines each, production-quality

## 🚀 Início Rápido Quick Start

### Instalar uma Habilidade Install a Skill

1. Copie `SKILL.md` para seu diretório de habilidades Claude | Copy `SKILL.md` to your Claude skills directory
2. Descreva o vídeo que deseja criar | Describe the video you want to create
3. Claude gera um prompt pronto para produção | Claude generates a production-ready prompt
4. Cole em [Seedance 2.0 (Higgsfield)](https://higgsfield.ai/create/video?model=seedance_2_0) | Paste into Seedance 2.0 on Higgsfield with your materials

### Exemplo Example

```
Você: Preciso de um vídeo cinemático de 15 segundos de um samurai solitário caminhando por uma floresta de bambu enevoada ao amanhecer
You: I need a 15s cinematic video of a lone samurai walking through foggy bamboo forest at dawn

Claude (usando habilidade cinemática with cinematic skill):
→ Gera um prompt detalhado de 25+ linhas Generates 25+ line detailed prompt
→ Com linha do tempo, câmera, iluminação, som, gancho de 2 segundos With timeline, camera, lighting, sound, 2-second hook
```

## 📋 Especificações Seedance 2.0 (Higgsfield) Specs

| Entrada Input | Formato Format | Limite Limit |
|---|---|---|
| Imagem Image | jpeg, png, webp, bmp, tiff, gif | ≤ 9, cada um < 30MB |
| Vídeo Video | mp4, mov | ≤ 3, cada um < 50MB, duração total 2–15s |
| Áudio Audio | mp3, wav | ≤ 3, cada um < 15MB, duração total ≤ 15s |
| Texto Text | Linguagem natural Natural language | — |
| **Combinado Combined** | — | **≤ 12 arquivos files** |
| **Saída Output** | Vídeo Video | **4–15s, 720p, com som with sound** |
