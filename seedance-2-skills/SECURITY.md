## 报告漏洞 Reporting a Vulnerability

本仓库包含的是 Claude 技能文件（Markdown 文本），不包含可执行代码或 API 密钥。安全风险较低，但如果你发现任何问题，请通过以下方式报告：

This repository contains Claude skill files (Markdown text) — no executable code or API keys. Security risk is low, but if you find any issues, please report via:

1. **GitHub Issues** — 用于非敏感问题 | For non-sensitive issues
2. **𝕏 DM @higgsfield** — 用于敏感问题 | For sensitive issues

### 我们关注的问题 What We Care About

- 技能文件中意外包含的 API 密钥或凭据 | Accidentally included API keys or credentials in skill files
- 提示词注入风险（技能内容可能被滥用）| Prompt injection risks (skill content that could be misused)
- 第三方链接被篡改 | Compromised third-party links

- 修复计划 Fix plan: 7 天内 | within 7 days
