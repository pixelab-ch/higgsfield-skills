# 이것은 무엇인가요? What Is This?

Seedance 2.0(Higgsfield)용으로 특별히 만들어진 15개의 전문 Claude 기술입니다. 각 기술은 Claude를 특정 비디오 스타일의 프롬프트 엔지니어로 변환하여 크고 상세하며 바로 사용 가능한 프롬프트를 생성합니다. 강력한 **2초 훅**을 포함하여 시청자의 스크롤을 멈춥니다.

15 specialized Claude skills built for Seedance 2.0 on Higgsfield. Each skill turns Claude into a prompt engineer for a specific video style — generating large, detailed, paste-ready prompts with powerful **2-second hooks** that stop the scroll.

## 각 기술에 포함된 내용 What Each Skill Contains

- **2초 훅 프레임워크 / 2-Second Hook Framework** — 10-12가지 주목을 끄는 오프닝 패턴 | 10-12 attention-grabbing opener patterns
- **타임라인 세분화 / Timeline Segmentation** — 비트별 분해, 최대 15초 | Beat-by-beat breakdown up to 15s
- **카메라 무브먼트 백과사전 / Camera Movement Encyclopedia** — 15-20개 이상의 기술과 정확한 프롬프트 표현 | 15-20+ techniques with exact phrasing
- **조명과 분위기 / Lighting & Atmosphere** — 분위기와 품질을 전달하는 조명 설정 | Setups that communicate mood and quality
- **사운드 디자인 / Sound Design** — 앰비언트, 폴리, 뮤직, 사일런스 | Ambient, foley, music, silence
- **자료 참고 전략 / Material Reference Strategy** — `@image1` `@video1` `@audio1`의 모범 사례 | Best practices for references
- **플랫폼 최적화 / Platform Optimization** — 더우인, TikTok, Instagram, YouTube 등 | Cross-platform adjustments
- **5개 이상의 대규모 프롬프트 예제 / 5+ Large Example Prompts** — 각 15-25줄, 프로덕션 수준 품질 | 15-25 lines each, production-quality

### 기술 설치 Install a Skill

1. `SKILL.md`를 Claude 기술 디렉터리에 복사합니다 | Copy `SKILL.md` to your Claude skills directory
2. 만들고 싶은 비디오를 설명합니다 | Describe the video you want to create
3. Claude가 프로덕션 수준의 프롬프트를 생성합니다 | Claude generates a production-ready prompt
4. [Seedance 2.0(Higgsfield)](https://higgsfield.ai/create/video?model=seedance_2_0)에 붙여넣습니다 | Paste into Seedance 2.0 on Higgsfield with your materials

```
당신: 새벽 안개 어린 대나무 숲을 걷는 고독한 사무라이의 15초 시네마틱 비디오가 필요합니다
You: I need a 15s cinematic video of a lone samurai walking through foggy bamboo forest at dawn

Claude (시네마틱 기술 사용 with cinematic skill):

→ 타임라인, 카메라, 조명, 사운드, 2초 훅을 포함합니다 With timeline, camera, lighting, sound, 2-second hook
```

## 📋 Seedance 2.0(Higgsfield) 사양 Specs

| 입력 Input | 형식 Format | 제한 Limit |
|---|---|---|
| 이미지 Image | jpeg, png, webp, bmp, tiff, gif | ≤ 9개, 각각 < 30MB |
| 비디오 Video | mp4, mov | ≤ 3개, 각각 < 50MB, 총 시간 2–15s |
| 오디오 Audio | mp3, wav | ≤ 3개, 각각 < 15MB, 총 시간 ≤ 15s |
| 텍스트 Text | 자연어 Natural language | — |
| **합계 Combined** | — | **≤ 12개 파일 files** |
| **출력 Output** | 비디오 Video | **4–15s, 720p, 사운드 포함 with sound** |
